# Mega Account Renamer Bot

## Setup
1. Clone repo
2. Run `npm install`
3. Create `.env` file:
   BOT_TOKEN=your_telegram_bot_token_here
4. Run `npm start`

## Commands
- /setmega <email> <password> - Login to Mega
- /rename_path <path> <new_name> - Rename a single file/folder
- /rename_all ... - Bulk rename
